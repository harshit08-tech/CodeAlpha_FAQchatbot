# 🤖 FAQ Chatbot

## 📌 Description
This project is developed as part of my Artificial Intelligence Internship at CodeAlpha.

The FAQ Chatbot is a simple rule-based chatbot that uses Natural Language Processing (NLP) techniques to answer user queries based on predefined questions.

---

## 🧠 How It Works
- The chatbot uses **TF-IDF (Term Frequency - Inverse Document Frequency)** to convert text into numerical form.
- It then uses **Cosine Similarity** to compare the user’s question with stored FAQ questions.
- The most similar question is selected, and its answer is displayed.

---

## 🛠️ Technologies Used
- Python
- Streamlit
- Scikit-learn
- NLP (TF-IDF & Cosine Similarity)

---

## ⚙️ Features
- Simple and interactive chatbot UI  
- Intelligent question matching using NLP  
- Fast response system  
- Input validation with fallback message
